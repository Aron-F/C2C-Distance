C2c-distance

Calculates the distance from one character on the keyboard to another character found in a "match" eg. when searching for something.

View on GitHub Download .zip Download .tar.gz

About C2C

It measures distance between keys on the keyboard.

How it's supposed to work.

The idea is that a user inputs a word, if that word is misspelled when compared to say a keyword that you have stored somewhere, you get a number representing the distance that the input word has from the keyword.
